# Centennial Connection

> A production-oriented social platform for students, built from scratch to learn modern software engineering while creating a real-world product.

---

## About the Project

Centennial Connection is a mobile application designed to help students connect, communicate, and build meaningful relationships within their educational community.

This repository contains the complete redevelopment of the project from the ground up. The goal is not only to build a functional application, but also to follow a professional software engineering process from product planning and system design to implementation, testing, deployment, and long-term maintenance.

Instead of relying on backend-as-a-service platforms, the application is built using a custom backend, relational database, and production-inspired architecture.

Every major engineering decision is documented throughout the development process.

---

## Vision

The long-term vision of Centennial Connection is to become a modern student platform that combines communication, discovery, communities, clubs, events, and personalized campus experiences into a single application.

Development follows an incremental approach, beginning with a minimal but production-quality foundation and expanding through additional features over time.

---

## Development Philosophy

This project follows several core principles:

- Plan before implementation.
- Build production-ready architecture.
- Keep the codebase modular and maintainable.
- Document important engineering decisions.
- Develop features incrementally.
- Prioritize long-term maintainability over short-term speed.
